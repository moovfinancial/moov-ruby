# DomesticPullFromCard

Indicates if the card supports domestic pull-from-card transfer.

## Example Usage

```ruby
require "moov_ruby"

value = DomesticPullFromCard::NOT_SUPPORTED
```


## Values

| Name            | Value           |
| --------------- | --------------- |
| `NOT_SUPPORTED` | not-supported   |
| `SUPPORTED`     | supported       |
| `UNKNOWN`       | unknown         |
