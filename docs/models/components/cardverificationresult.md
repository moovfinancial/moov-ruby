# CardVerificationResult

The result of a card verification check.

## Example Usage

```ruby
require "moov_ruby"

value = CardVerificationResult::NO_MATCH
```


## Values

| Name            | Value           |
| --------------- | --------------- |
| `NO_MATCH`      | noMatch         |
| `MATCH`         | match           |
| `NOT_CHECKED`   | notChecked      |
| `UNAVAILABLE`   | unavailable     |
| `PARTIAL_MATCH` | partialMatch    |
