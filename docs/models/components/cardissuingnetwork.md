# CardIssuingNetwork

The name of the network a card transaction is routed through.

## Example Usage

```ruby
require "moov_ruby"

value = CardIssuingNetwork::DISCOVER
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `DISCOVER` | discover   |
| `SHAZAM`   | shazam     |
| `VISA`     | visa       |
