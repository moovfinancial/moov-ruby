# FeeModel

Specifies the pricing model used for the calculation of the final fee.

## Example Usage

```ruby
require "moov_ruby"

value = FeeModel::FIXED
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `FIXED`    | fixed      |
| `BLENDED`  | blended    |
| `VARIABLE` | variable   |
