# AuthMethod

The authentication method used for the Google Pay token.

## Example Usage

```ruby
require "moov_ruby"

value = AuthMethod::PAN_ONLY
```


## Values

| Name              | Value             |
| ----------------- | ----------------- |
| `PAN_ONLY`        | PAN_ONLY          |
| `CRYPTOGRAM_3_DS` | CRYPTOGRAM_3DS    |
