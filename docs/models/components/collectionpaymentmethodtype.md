# CollectionPaymentMethodType

Payment methods allowed for collecting a payment.

## Example Usage

```ruby
require "moov_ruby"

value = CollectionPaymentMethodType::APPLE_PAY
```


## Values

| Name                | Value               |
| ------------------- | ------------------- |
| `APPLE_PAY`         | apple-pay           |
| `CARD_PAYMENT`      | card-payment        |
| `ACH_DEBIT_COLLECT` | ach-debit-collect   |
