# InstantBankTransactionStatus

Status of a transaction within the instant-bank lifecycle.

## Example Usage

```ruby
require "moov_ruby"

value = InstantBankTransactionStatus::INITIATED
```


## Values

| Name                       | Value                      |
| -------------------------- | -------------------------- |
| `INITIATED`                | initiated                  |
| `COMPLETED`                | completed                  |
| `FAILED`                   | failed                     |
| `ACCEPTED_WITHOUT_POSTING` | accepted-without-posting   |
