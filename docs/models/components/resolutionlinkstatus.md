# ResolutionLinkStatus

The status of a resolution link.

## Example Usage

```ruby
require "moov_ruby"

value = ResolutionLinkStatus::ACTIVE
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `ACTIVE`    | active      |
| `SUBMITTED` | submitted   |
| `COMPLETED` | completed   |
| `DISABLED`  | disabled    |
| `EXPIRED`   | expired     |
