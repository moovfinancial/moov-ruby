# CardFundingSource

The funding source of the card. One of CREDIT, DEBIT, PREPAID, or UNKNOWN.

## Example Usage

```ruby
require "moov_ruby"

value = CardFundingSource::CREDIT
```


## Values

| Name      | Value     |
| --------- | --------- |
| `CREDIT`  | CREDIT    |
| `DEBIT`   | DEBIT     |
| `PREPAID` | PREPAID   |
| `UNKNOWN` | UNKNOWN   |
