# TerminalApplicationStatus

Status of the terminal application.

## Example Usage

```ruby
require "moov_ruby"

value = TerminalApplicationStatus::UNDEFINED
```


## Values

| Name        | Value       |
| ----------- | ----------- |
| `UNDEFINED` | undefined   |
| `PENDING`   | pending     |
| `ENABLED`   | enabled     |
| `DISABLED`  | disabled    |
